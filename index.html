# [ GIT BASICS ] (CheatSheet)

By: Your Name

1. Repository Operations

  • Initialize a repository: git init
  • Clone a repository: git clone <repository_url>
  • Check repository status: git status
  • List all remotes: git remote -v
  • Add a remote: git remote add <name> <url>

2. Branch Operations

  • Create a branch: git branch <branch_name>
  • Switch to a branch: git checkout <branch_name>
  • Create and switch to branch: git checkout -b <branch_name>
  • List all branches: git branch -a
  • Delete a branch: git branch -d <branch_name>
  • Merge a branch: git merge <branch_name>
  • Rebase current branch: git rebase <branch_name>

3. Commit Operations

  • Stage changes: git add <file> or git add .
  • Commit changes: git commit -m "commit message"
  • Amend last commit: git commit --amend
  • View commit history: git log
  • View specific commit: git show <commit_hash>
  • Reset to previous commit: git reset <commit_hash>

4. Remote Operations

  • Push to remote: git push <remote> <branch>
  • Pull from remote: git pull <remote> <branch>
  • Fetch from remote: git fetch <remote>
  • View remote branches: git branch -r
  • Track remote branch: git branch --track <branch> <remote>/<branch>

5. Stash Operations

  • Stash changes: git stash
  • List stashes: git stash list
  • Apply stash: git stash apply
  • Drop stash: git stash drop
  • Clear all stashes: git stash clear

6. Tag Operations

  • Create a tag: git tag <tag_name>
  • Create annotated tag: git tag -a <tag_name> -m "message"
  • List tags: git tag
  • Push tags to remote: git push --tags
  • Delete tag: git tag -d <tag_name>

7. Configuration

  • Set global username: git config --global user.name "Your Name"
  • Set global email: git config --global user.email "your@email.com"
  • View configuration: git config --list
  • Set default editor: git config --global core.editor "code --wait"

8. Advanced Operations

  • Cherry-pick a commit: git cherry-pick <commit_hash>
  • Interactive rebase: git rebase -i <commit_hash>
  • Search in code: git grep "search_term"
  • Show file history: git log -p <file_path>
  • Create a patch: git format-patch <commit_hash>

9. Undoing Changes

  • Unstage a file: git reset HEAD <file>
  • Discard local changes: git checkout -- <file>
  • Revert a commit: git revert <commit_hash>
  • Reset to remote: git fetch origin && git reset --hard origin/<branch>

10. Submodules

  • Add a submodule: git submodule add <repository_url> <path>
  • Initialize submodules: git submodule init
  • Update submodules: git submodule update
  • Clone with submodules: git clone --recurse-submodules <repository_url>
